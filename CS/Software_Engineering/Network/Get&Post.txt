[Spring] GetMapping & PostMapping의 차이
GET 방식은 어떠한 정보를 가져와서 조회하기 위해 사용되는 방식,
POST 방식은 데이터를 서버로 제출하여 추가/수정하기 위해 데이터를 전송하는 방식이다.

GET과 POST방식의 차이는 다음과 같다.
GET
- URL에 변수를 포함시켜 요청
- 데이터를 header에 포함하여 전송
- URL에 데이터가 노출되어 보안에 취약
- 캐싱 가능
POST
- URL에 변수를 노출 X
- 데이터 body에 포함
- URL에 데이터가 노출안되어 기본 보안이 설정되어 있음
- 전송 길이 제한 X
- 캐싱 불가능
